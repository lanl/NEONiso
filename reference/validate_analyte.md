# Standardize analyte names

Enforces standard capitalization and formatting of H2o and Co2 analyte
names across calibration functions.

## Usage

``` r
validate_analyte(analyte)
```

## Arguments

- analyte:

  Co2 or H2o?

## Value

Standardized string for the water ('H2o') or carbon ('Co2') systems to
make sure strings are standardized across package functions.
