# List terrestrial gradient sites

Returns a vector of four-letter NEON site codes for the gradient
terrestrial sites that have TIS instrumentation.

## Usage

``` r
terrestrial_gradient_sites()
```

## Value

A vector listing NEON gradient terrestrial sites.

## Author

Rich Fiorella <rfiorella@lanl.gov>

## Examples

``` r
terrestrial_gradient_sites()
#>  [1] "ABBY" "BARR" "BART" "BLAN" "DCFS" "DEJU" "DELA" "DSNY" "GRSM" "HEAL"
#> [11] "JERC" "JORN" "KONA" "LAJA" "LENO" "MLBS" "MOAB" "NOGP" "OAES" "RMNP"
#> [21] "SERC" "SOAP" "STEI" "STER" "TEAK" "TREE" "UKFS"
```
