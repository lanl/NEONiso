# List terrestrial core sites

Returns a vector of four-letter NEON site codes for the core terrestrial
sites that have TIS instrumentation.

## Usage

``` r
terrestrial_core_sites()
```

## Value

A vector listing NEON core terrestrial sites.

## Author

Rich Fiorella <rfiorella@lanl.gov>

## Examples

``` r
terrestrial_core_sites()
#>  [1] "BONA" "CLBJ" "CPER" "GUAN" "HARV" "KONZ" "NIWO" "ONAQ" "ORNL" "OSBS"
#> [11] "PUUM" "SCBI" "SJER" "SRER" "TALL" "TOOL" "UNDE" "WOOD" "WREF" "YELL"
```
