# List sites with water vapor isotope ratios.

Returns a vector of four-letter NEON site codes for the terrestrial
sites that have water vapor isotope ratio instrumentation.

## Usage

``` r
water_isotope_sites()
```

## Value

A vector listing NEON sites measuring water vapor isotope ratios.

## Author

Rich Fiorella <rfiorella@lanl.gov>

## Examples

``` r
water_isotope_sites()
#>  [1] "BONA" "CLBJ" "CPER" "GUAN" "HARV" "KONZ" "NIWO" "ONAQ" "ORNL" "OSBS"
#> [11] "PUUM" "SCBI" "SJER" "SRER" "TALL" "TOOL" "UNDE" "WOOD" "WREF" "YELL"
#> [21] "BARR"
```
